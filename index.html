Get started

1

Prerequisites

Install Node.js 20+ and Claude Code. Then run claude in your terminal and sign in. Claude Code requires a Claude Pro or Max subscription.

Already have both? Skip to step 2.

2

Install and start

Run these in your terminal. This creates a rundock folder in your current directory.

git clone https://github.com/liamdarmody/rundock.git cd rundock npm install npm start

To update later: npm run update

3

Open Rundock

Open http://localhost:3000 in your browser. Choose a workspace folder or create a new one. A workspace is any folder where you keep your work: documents, research, projects. This should be separate from the Rundock install folder.

4

Build your team

Doc, the built-in guide, analyses your workspace and proposes an agent team. Say go, and your agents appear on the org chart. Start a conversation and put them to work.
